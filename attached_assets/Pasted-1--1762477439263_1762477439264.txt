1. 전체 개요

본 시스템은 누수 사고의 접수부터 현장조사, 견적, 정산, 통계, 기준정보 관리까지 전 과정을 디지털화한 통합 관리 플랫폼입니다.
보험사, 플랫폼 관리자(플록슨), 수리업체(협력사) 간의 업무 흐름을 자동화하고, 보고서·견적서·정산서 등의 문서를 시스템 내에서 생성·전송·승인할 수 있도록 지원합니다.

2. 주요 기능 / 페이지별 설명
① 홈(Home)

누가: 모든 사용자 (플랫폼 담당자, 협력업체, 보험사)

목적: 전체 진행 현황 및 주요 지표 확인, 빠른 메뉴 이동

표시 정보: 접수건수, 진행률, 미정산 내역, 알림, 문의사항 등

액션: 주요 모듈 바로가기, 1:1 문의 작성, 실시간 알림 확인

② 접수 관리 (Reception Management)

누가: 플랫폼 담당자

목적: 보험사로부터 접수된 누수 사고 신규 등록 및 관리

입력 정보:
접수번호(자동), 접수일자, 보험사명, 담당자, 연락처, 사고 원인, 피해유형, 추정손해액, VIP 여부, 피해 위치(주소 검색), 피해 품목/수량 등

액션:

신규등록 / 저장 / 초기화

협력업체 배정(출동가능권역 필터 기반)

배당 협력사 변경 팝업 (REQM-0106)

특수기능:

협력업체 배당 시 “출동 가능 권역”에 해당하는 업체 리스트 자동 표시

권역별 대체 협력업체 선택 팝업 제공

③ 현장 조사 (Field Survey)

누가: 수리업체 사용자

목적: 현장 방문 결과 및 조사 내용 입력

입력 정보: 사고 원인, 이동거리, 특이사항, VOC, 복구 방식(부분/전체), 손해액, 견적정보

액션: 저장 / 수정 / 보고서용 데이터 자동 연동

④ 도면 작성 (Restoration Estimation)

누가: 수리업체 / 관리자

목적: 피해 범위 도면 기반 기록 및 복구 범위 산정

입력 정보: 공간별 피해 위치, 오염 지역, 원인 지점 등

액션:

디지털 드로잉(원인, 피해, 오염 구역 표시)

저장 시 견적서·보고서 자동 연동

⑤ 이미지 및 파일 관리

누가: 수리업체 / 관리자

목적: 단계별 이미지 및 파일 업로드·관리

입력 정보:
초기/중간/완성 이미지, 청구 증빙서류, 동영상 등

액션: 업로드 / 다운로드 / 미리보기 / 일괄관리

권한: 관련 케이스별로 접근권한 제한 적용

⑥ 견적서 관리 (Estimate)

누가: 수리업체

목적: 복구비용 견적 작성 및 제출

입력 정보: 공종, 자재, 규격, 단가, 수량, VAT, 합계

액션:

자동 합계 계산

PDF / Excel 다운로드

제출 후 심사자 검토

특수기능:

현장 피해면적표 자동 생성

소계, 일반관리비(6%), 이윤(15%), VAT 자동 계산

⑦ 현장출동보고서 (Reports)

누가: 관리자 / 보험사

목적: 종합 보고서 생성 및 보험사 제출

표시 정보: 접수·조사·피해·견적 요약, 중복보험 정보

액션:

PDF 다운로드, 이메일 발송

중복보험 관리(추가/삭제/안분조회)

첨부 파일 (REQM-0706):

수리중/완료 사진

복구완료확인서, 도급계약서, 개인정보동의서 등

1차 현장보고서 누락 서류 자동 포함

금액 및 수수료는 접수/승인 내역에서 자동 불러오기

⑧ 진행 관리 (Progress)

누가: 관리자 / 수리업체

목적: 케이스별 진행 단계 추적 및 승인/반려 관리

표시 정보: 진행상태(접수 → 심사대기 → 승인/반려 등), 담당자, 사고번호, 주요 진행사항

액션:

상태 변경, 심사 승인/반려, 진행이력 기록

알림 발송 (문자 메시지 포함)

추가 기능:

INVOICE 발행 및 이메일 발송 (REQM-0706)

자동 첨부파일 포함

⑨ 통계 및 정산 (Finance & Settlement)

누가: 관리자 / 경영진

목적: 통계 조회, 정산/미수금 관리, 입금현황 매칭

표시 정보: 건별 통계, 회계현황, 예상손해액, 미수금 리스트

액션:

Excel 다운로드, 필터 조회

입금내역 업로드 후 팝업에서 수동 매칭(REQM-0803)

매칭 시 “미정산 → 정산” 상태 자동 변경

⑩ 관리자 메뉴 (Admin / Master)

누가: 플랫폼 관리자

목적: 기준정보, 사용자, 권한, 코드 관리 및 문자 발송 관리

세부 기능:

사용자/역할/권한 관리

코드(피해유형, 사고원인 등) 관리

협력업체 등록 및 서류(사업자등록증, 신분증, 통장사본) 관리

공종DB 관리(노무비/자재비 DB)
